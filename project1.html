Space Radiation Testing System

MATLAB Image Processing Automation Project Management

This project is a low-dose radiation testing system designed to test robustness and performance of a camera's image quality, current consumption, etc over time.

Overview

The system is programmed to automatically and remotely take images of a template picture with controlled lighting. The images were collected and processed in a MATLAB script to plot changes in image quality over time.

Key Features

Automated Image Capture

MATLAB script that takes pictures and records data from camera automatically

Controlled Lighting

Controlled LED-lighting box for consistency

Radiation Chamber

Co60 chamber with controlled radiation dose over time

Technical Challenges

One of the main challenges was ensuring consistent lighting conditions. This was solved by installing LEDs, camera, and image on a secure container.

Results

The project provided valuable data for MDA space through a publication in IEEE Xplore.
